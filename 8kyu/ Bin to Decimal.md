## Bin to Decimal - 8kyu

Complete the function which converts a binary number (given as a string) to a decimal number.



### Given Code


```python
def bin_to_decimal(inp):
    pass
```

---

### Solution


```python
def bin_to_decimal(inp):
    return int(inp,2)
```
